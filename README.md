# Phishing Awareness Project
## Description
This project demonstrates how phishing attacks work and how users can identify and prevent them. It includes a simulated login page to show how attackers capture sensitive information.
## Objectives
- Understand phishing attacks
- Identify red flags in phishing messages
- Demonstrate credential capture using a fake login page
- Spread awareness about cybersecurity
## Technologies Used
- HTML
- JavaScript
## Features
- Fake bank login page simulation
- Captures entered username and password (demo purpose only)
- Shows how phishing tricks users
## Disclaimer
This project is created for educational purposes only. It does not store or misuse any user data.
## Files Included
- demo.html → phishing login page
- Phishing report.pdf → detailed project report
## How to Run
1. Download the project
2. Open demo.html in browser
3. Enter any username and password
4. See how data is captured (demo)
